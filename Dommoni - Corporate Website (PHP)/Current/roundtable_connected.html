RoundTable enables great things to happen when run standalone. However, by creating components that run within the software you use everyday and by enabling that software to connect to and take advantage of the RoundTable Environment, that software suddenly becomes significantly more useful and more powerful. This is because what was an isolated, standalone application is now part of a comprehensive collaboration environment. So, anything you or members of your team build with those tools can now be more fluidly shared and communicated.

 

Imagine…
Imagine being able to capture the tasks the members of your team have created weather they where originally created as a source-code comment in Eclipse or Visual Studio, a task created in Microsoft Outlook, or a ToDo item created in the RoundTable web interface itself, imagine being able to bring all of those tasks together and automatically from them creating a project schedule in Microsoft Project! That's the power and the kinds of things you can do with RoundTable Connected Services.
